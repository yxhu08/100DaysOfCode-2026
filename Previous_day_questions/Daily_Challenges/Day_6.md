## 1. House Robber II (LeetCode #213)
A problem that teaches dynamic programming and optimization techniques by finding the maximum amount of money that can be robbed without robbing adjacent houses in a circular arrangement. Given an array where each element represents the amount of money in a house, return the maximum amount you can rob without robbing two adjacent houses. Since the houses are arranged in a circle, the first and last houses are also adjacent.

This problem is commonly asked in interviews and helps build concepts like:
- Circular arrays
- State optimization
- Decision making

which are important for solving optimization problems efficiently.

**Your task:** Find the maximum amount that can be robbed using an optimized approach.

### Examples

**Input:**
```
nums = [2, 3, 2]
```
**Output:**
```
3
```

---

**Input:**
```
nums = [1, 2, 3, 1]
```
**Output:**
```
4
```

---

**Input:**
```
nums = [1, 2, 3]
```
**Output:**
```
3
```

## 2. Union of Two Sorted Arrays
A problem that teaches two-pointer and array merging techniques by finding all distinct elements present in two sorted arrays. Given two sorted arrays a[] and b[], return the union of both arrays in sorted order. The union should contain only distinct elements, even if duplicate values are present in the arrays.

This problem is commonly asked in interviews and helps build concepts like:
- Sorted array traversal
- Duplicate handling
- Merging techniques

which are important for efficient array-processing problems.

**Your task:** Find the union of two sorted arrays using an optimized approach.

### Examples

**Input:**
```
a[] = [1, 2, 3, 4, 5], b[] = [1, 2, 3, 6, 7]
```
**Output:**
```
[1, 2, 3, 4, 5, 6, 7]
```

---

**Input:**
```
a[] = [2, 2, 3, 4, 5], b[] = [1, 1, 2, 3, 4]
```
**Output:**
```
[1, 2, 3, 4, 5]
```

---

**Input:**
```
a[] = [1, 1, 1, 1, 1], b[] = [2, 2, 2, 2, 2]
```
**Output:**
```
[1, 2]
```
