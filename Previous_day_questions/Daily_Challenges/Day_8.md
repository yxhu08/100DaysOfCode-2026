## 1. Valid Palindrome (LeetCode #125)
A problem that focuses on string processing and character validation by determining whether a string reads the same forward and backward after preprocessing. Given a string s, determine whether it is a palindrome after converting uppercase letters to lowercase and removing all non-alphanumeric characters. Return true if the string is a palindrome; otherwise return false.

This problem is commonly asked in interviews and helps build concepts like:
- String processing
- Character validation
- Efficient string traversal
- Case conversion

which are important for text-processing problems.

**Your task:** Check whether the given string is a valid palindrome using an optimized approach.

### Examples

**Input:**
```
s = "A man, a plan, a canal: Panama"
```
**Output:**
```
true
```

---

**Input:**
```
s = "race a car"
```
**Output:**
```
false
```

---

**Input:**
```
s = " "
```
**Output:**
```
true
```

---

## 2. Reverse a String
A problem that focuses on basic string manipulation by reversing the characters of a given string. Given a string s, reverse the string and return the reversed result.

This problem is commonly asked in interviews and helps build concepts like:
- String traversal
- Character swapping
- In-place modification
- Basic string manipulation

which are important for solving many string-based programming problems.

**Your task:** Reverse the given string using an optimized approach.

### Examples

**Input:**
```
s = "Computer"
```
**Output:**
```
"retupmoC"
```

---

**Input:**
```
s = "for"
```
**Output:**
```
"rof"
```

---

**Input:**
```
s = "a"
```
**Output:**
```
"a"
```
