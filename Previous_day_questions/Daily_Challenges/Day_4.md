## 1. Transpose Matrix (LeetCode #867)
A problem that teaches matrix traversal and in-place swapping techniques by converting rows into columns.

Given a 2D integer matrix, return its transpose. The transpose of a matrix is formed by converting all rows into columns and all columns into rows.

This problem is commonly asked in interviews and helps build concepts like:
- Matrix traversal
- Nested loops
- Row-column transformation
- 2D array manipulation

which are important for efficient matrix-processing problems.

**Your task:** Transpose the given matrix using an optimized approach.

### Examples

**Input:**
```
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```
**Output:**
```
[[1, 4, 7], [2, 5, 8], [3, 6, 9]]
```

---

**Input:**
```
matrix = [[1, 2, 3], [4, 5, 6]]
```
**Output:**
```
[[1, 4], [2, 5], [3, 6]]
```

## 2. Rotate Array by K Steps
A problem that teaches array manipulation and rotation techniques by shifting elements of an array by a given number of positions.

Given an array of integers and an integer k, rotate the array to the right by k steps. Elements shifted beyond the end of the array should wrap around to the beginning.

This problem is commonly asked in interviews and helps build concepts like:
- Array manipulation
- Index calculation
- Modular arithmetic
- Optimization techniques

which are important for efficient array-processing problems.

**Your task:** Rotate the given array by k steps using an optimized approach.

### Examples

**Input:**
```
Array = [1, 2, 3, 4, 5, 6, 7], k = 3
```
**Output:**
```
[5, 6, 7, 1, 2, 3, 4]
```

---

**Input:**
```
Array = [-1, -100, 3, 99], k = 2
```
**Output:**
```
[3, 99, -1, -100]
```
