## 1. Jump Game (LeetCode #55)
A problem that teaches traversal techniques by determining whether it is possible to reach the last index of an array. Given an array where each element represents the maximum jump length from that position, determine if you can reach the last index starting from the first index.

This problem is commonly asked in interviews and helps build concepts like:
- Array traversal
- Reachability problems
- Optimization techniques

which are important for efficient decision-making problems.

**Your task:** Determine whether the last index can be reached using an optimized approach.

### Examples

**Input:**
```
nums = [2, 3, 1, 1, 4]
```
**Output:**
```
true
```

---

**Input:**
```
nums = [3, 2, 1, 0, 4]
```
**Output:**
```
false
```

## 2. Check Equal Arrays
A problem that teaches array comparison and frequency checking techniques by determining whether two arrays contain the same elements. Given two arrays a[] and b[] of equal size, check whether both arrays contain the same elements. The order of elements may be different, but the frequency of each element must be the same.

This problem is commonly asked in interviews and helps build concepts like:
- Frequency counting
- Array traversal
- Comparison techniques

which are important for data validation and matching problems.

**Your task:** Determine whether the two arrays are equal using an optimized approach.

### Examples

**Input:**
```
a[] = [1, 2, 5, 4, 0], b[] = [2, 4, 5, 0, 1]
```
**Output:**
```
true
```

---

**Input:**
```
a[] = [1, 2, 5], b[] = [2, 4, 15]
```
**Output:**
```
false
```
